# NOUS - The ultimate keyboard
Nor(wegian|dic) + United States
Made with scandinavian keyboards in mind. Uses both Shift and AltGr to switch keys.

# Installation
```sh
# cp nous /usr/share/X11/xkb/symbols
$ setxkbmap nous
```
